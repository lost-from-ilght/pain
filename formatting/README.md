# Code Formatter

This directory contains the code formatting tools for the Fansocial Admin project.

## Quick Start

**Windows**: Double-click `format.bat` to format all code files.

**Manual**:
```bash
cd formatting
npm install
npm run format
```

## Documentation

See [FORMATTING_INSTRUCTIONS.md](./FORMATTING_INSTRUCTIONS.md) for complete coding standards and formatting guidelines.

## What It Does

- Formats all JavaScript, HTML, CSS, and JSON files
- Creates automatic backups before formatting
- Applies consistent indentation, spacing, and style
- Follows industry-standard formatting rules

## Files

- `format.bat` - One-click formatter (Windows)
- `gulpfile.js` - Gulp tasks for formatting
- `.prettierrc` - Prettier configuration
- `package.json` - Dependencies and scripts
- `FORMATTING_INSTRUCTIONS.md` - Complete coding standards guide
